### Linux Fundamentals
-------------------------
pwd - print where directory

ls - list

li - l - list alphabetical order

li - lr - reverse list alphabetical order

mkdir - make directory Ex- mkdir alien

touch - create txt, yml etc files Ex- touch alien.txt

rm - remove empty file/ directory Ex- rm alien.txt

rm -rf - remove files/ directory with content , force delete Ex- rm alien.txt(with content)

cat - to add content in files 

cat >> - to upldate content
